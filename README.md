# golang-dtqs
A distributed task queue system built with Go and Redis.

## Planned Features

- Distributed task processing
- Priority queues
- Automatic retries
- REST API
- Horizontal scaling

## Architecture

Will use Redis for queue storage and Go workers for processing.
